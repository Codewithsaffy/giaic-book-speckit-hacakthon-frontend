## Usage

When implementing the landing page:

1. **Read the main prompt** (`landing-page-prompt.md`) for complete content and structure
2. **Review all design reference images** to understand the visual aesthetic and layout patterns
3. **Follow the design principles** outlined in the implementation notes
4. **Maintain consistency** with existing site theme (colors, fonts)
5. **Prioritize mobile-first** responsive design

## Key Design Principles

- Dark theme with red/purple gradients
- Glassmorphism effects for modern feel
- Smooth animations and micro-interactions
- Clean typography with proper hierarchy
- Fast loading and optimized performance

## Notes

- All content should be adapted to match the actual book/documentation context
- Design references are for inspiration - adapt to fit the existing site theme
- Focus on creating a premium, modern feel that wows users at first glance
