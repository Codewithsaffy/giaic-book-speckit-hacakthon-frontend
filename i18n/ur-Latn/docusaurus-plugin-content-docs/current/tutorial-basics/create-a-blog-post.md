# Ek Blog Post Banayein

Docusaurus **har blog post ke liye ek page** banata hai, saath hi ek **blog index page**, ek **tag system**, ek **RSS** feed...

## Apni Pehli Post Banayein

`blog/2021-02-28-greetings.md` par ek file banayein:

```md title="blog/2021-02-28-greetings.md"
---
slug: greetings
title: Salaam!
authors:
  - name: Joel Marcey
    title: Docusaurus 1 ke Co-creator
    url: https://github.com/JoelMarcey
    image_url: https://github.com/JoelMarcey.png
  - name: SÃ©bastien Lorber
    title: Docusaurus maintainer
    url: https://sebastienlorber.com
    image_url: https://github.com/slorber.png
tags: [greetings]
---

Mubarak ho, aap ne apni pehli post bana li hai!

Is post ko jitna chahein edit karein aur khel lein.
```

Ab ek nayi blog post yahan available hai: [http://localhost:3000/blog/greetings](http://localhost:3000/blog/greetings)
