# Mubarak Ho!

Aap ne abhi **Docusaurus ki bunyadi baatein** seekh li hain aur **initial template** mein kuch tabdeeliyan ki hain.

Docusaurus ke paas **bahut kuch aur bhi hai**!

Kya aap ke paas **5 mazeed minute** hain? **[versioning](../tutorial-extras/manage-docs-versions.md)** aur **[i18n](../tutorial-extras/translate-your-site.md)** par nazar daalein.

Kuch **samajh nahi aaya** ya **bug** hai is tutorial mein? [Please report karein!](https://github.com/facebook/docusaurus/discussions/4610)

## Aage Kya Hai?

- [Official documentation](https://docusaurus.io/) padhein
- [`docusaurus.config.js`](https://docusaurus.io/docs/api/docusaurus-config) se apni site configuration modify karein
- [`themeConfig`](https://docusaurus.io/docs/api/themes/configuration) se navbar aur footer items add karein
- Custom [Design and Layout](https://docusaurus.io/docs/styling-layout) add karein
- [Search bar](https://docusaurus.io/docs/search) add karein
- [Docusaurus showcase](https://docusaurus.io/showcase) mein inspiration dhundhein
- [Docusaurus Community](https://docusaurus.io/community/support) mein shamil hon
