# Shuruat Karein

Chalein **Docusaurus ko 5 minute se kam mein** jaan lein.

## Shuruat Karein

**Nayi site bana kar** shuru karein.

Ya **Docusaurus ko turant azmayein** **[docusaurus.new](https://docusaurus.new)** ke saath.

### Aap ko kya chahiye hoga

- [Node.js](https://nodejs.org/en/download/) version 18.0 ya usse upar:
  - Jab Node.js install kar rahe hain, dependencies se mutalliq sab checkboxes check karna behtar hai.

## Nayi Site Generate Karein

**Classic template** use karke nayi Docusaurus site generate karein.

Classic template automatically aap ke project mein add ho jayega jab aap yeh command chalayein:

```bash
npm init docusaurus@latest my-website classic
```

Aap yeh command Command Prompt, Powershell, Terminal, ya apne code editor ke kisi integrated terminal mein type kar sakte hain.

Yeh command Docusaurus chalane ke liye zaroori sab dependencies bhi install kar deta hai.

## Apni Site Start Karein

Development server chalayein:

```bash
cd my-website
npm run start
```

`cd` command us directory ko change karta hai jis ke saath aap kaam kar rahe hain. Apni nayi Docusaurus site ke saath kaam karne ke liye, aap ko terminal ko wahan navigate karna hoga.

`npm run start` command aap ki website ko locally build karta hai aur use development server ke zariye serve karta hai, jo aap ke liye http://localhost:3000/ par dekhne ke liye tayyar hota hai.

`docs/intro.md` (yeh page) kholein aur kuch lines edit karein: site **automatically reload hoti hai** aur aap ki tabdeeliyan dikha deti hai.
